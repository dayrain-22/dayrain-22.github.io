manjaro archlinux

問題排除

manjaro更新時，停止在archlinux-keyring

狀況描述

執行 sudo pacman -Syu 時，停止在archlinux-

  1. 網路正常
  2. DNS解析正常
  3. DB也顯示為最新

但就是不會進行下載動作，在用archlinux沒遇過，順便記錄一下此問題
我想應該是因為在manjaro上，用了archlinux的mirrorlist site更新的關係

解決方式

  1. 清理keyring檔案
sudo rm /usr/share/pacman/keyrings/archlinux*
sudo rm -rf /etc/pacman.d/gnupg/*
sudo pacman-key --init
  1. 重新更新mirrorlist
sudo pacman-mirrors --country Taiwan,Japan,Singapore
  1. 再更新一次DB/Keyring
sudo pacman -Syy
sudo pacman-key --refresh-keys
sudo pacman-key --populate archlinux manjaro
  1. 重新更新package，因為archlinux與manjaro有版本差異，混合在一起很難說有什麼問題，最好是重新downgrade差異的package版本
