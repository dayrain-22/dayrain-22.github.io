基礎認知

  • process memory
    1. Code(.text) - 載入的二進制機器碼
    2. Data - 全域變數
    3. Heap 堆 - 動態請求大小空間使用/回收,malloc/free/delete...，由開發者控制
    4. Stack 棧 - 動態儲存函數之間呼叫關係，當呼叫其他函數後，以便返回時恢復母函數,由系統自動分配釋放
      1. 操作
        1. PUSH - 放入
        2. POP - 取出
      2. 位置標示
        1. TOP - 最上層位置
          1. ESP,Extended stack porinter
        2. BASE - 最下層位置
          1. EBP,Extended base porinter
      3. register暫存器
        1. EIP,,Extended Instruction porinter,指向下一個指令位置
  1. 程式碼內容-機械碼存入.text
  2. 引入函數表-裝載器填充函數,讓程式碼能取得函數位置
  3. optional header-讓裝載器讀取的結構,修正動態模組

.text 程式節，機械碼程式
.data 資料節，程式的全域變數和靜態變數，巨集定義
.rsrc 資源節，程式資源
